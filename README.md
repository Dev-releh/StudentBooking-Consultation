# Student Booking Consultation System – Business Analysis Project

## 📌 Project Overview
A complete business analysis for a web-based student consultation booking system that enables students to book, track, and manage academic consultations with advisors. This was a **self-initiated independent project**.I identified the gap, defined the scope and completed all analysis from start to finish.

---

## 📊 Key Highlights
- 📄 **26 functional and 16 non-functional requirements** documented
- 📝 **37 Agile user stories** written with acceptance criteria
- 🔍 **15 critical gaps** identified (missing advisor role, no automated reminders, no audit trail)
- 🖼️ **18 prototype screenshots** analyzed
- 📋 **Complete Business Requirements Specification (BRS)** delivered
